## 使い方

### エディタの起動

1. `editor.html` をブラウザで開く
2. 初回は空の状態なので、「JSON読込」から `sample-data.json` を読み込む
3. カテゴリを選択してパーツを編集

### 基本操作

1. **カテゴリ追加**: 左カラム下部の「+ カテゴリ追加」
   - 複数選択モード（アクセサリー用）の設定可能
   - 条件付き表示カテゴリの設定可能
2. **パーツ追加**: カテゴリ選択後、中央カラムの「+ 新規パーツ」
3. **パーツ編集**: パーツカードの「編集」ボタン
4. **レイヤー管理**: レイヤー追加、並び替え、zIndex設定
5. **色設定**: 
   - 色変更可能チェックを入れる
   - ブレンドモード（multiply, overlay等）または専用画像を選択
   - カラープリセットを追加（black, brown等）
6. **依存関係**:
   - 必須パーツ（requires）: このパーツが必要
   - 表示するカテゴリ/パーツ（unlocks）: このパーツ選択時に表示
7. **プレビュー**: 
   - 右カラムのキャンバスで確認
   - 「他パーツと重ねて表示」で組み合わせ確認
8. **バリデーション**: 警告セクションで問題をチェック
9. **保存**: 右カラム下部の「パーツを保存」

### JSONファイルの保存

- ヘッダーの「JSON出力」ボタンで `parts-data.json` をダウンロード
- 「保存」ボタンでLocalStorageに自動保存（次回起動時に復元）
